##초기설정
  1. 현재 디렉토리에서 npm install
  2. upload 폴더 생성
  3. cd ./client -> npm install
  4. cd ../
  7. 아래DB설정
  6. npm run dev

##DB 설정
아래의 json 파일생성후 아래내용 기입   
__database.json__
<pre>
<code>
{   
	"host" : "주소",   
	"user" : "유저명",   
	"password" : "유저비밀번호",   
	"port" : "포트번호",
